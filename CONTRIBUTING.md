# Как внести вклад

## Как запустить проект
1. Клонируйте репозиторий.
2. Убедитесь, что установлен PHP 8.0+.

## Требования к коду
- Код должен соответствовать стандарту [PSR-12](https://www.php-fig.org/psr/psr-12/).
- Все методы сопровождаются PHPDoc-комментариями.

## Как отправлять pull request
1. Создайте ветку от `main`.
2. Реализуйте изменения.
3. Убедитесь, что проходят CI-проверки.
4. Откройте pull request в `main`.

## Как писать коммиты
- Коммиты на английском.
- Используйте [Conventional Commits](https://www.conventionalcommits.org/): `feat:`, `fix:`, `chore:`, `docs:`, и т.д.
